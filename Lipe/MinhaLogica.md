# Lógica do Code forces numero: 1899A - Game with Integers

A solução usa o operador modulo (%) para verificar se o número é ou não divisível por 3.
Se o numero for divisível por 3, o primeiro jogador não consegue vencer automáticamente, pois qualquer outra jogada dele fará o numero deixar de ser divisível por 3.
Caso o numero não seja divisível por 3, o jogador pode somar ou subtrair 1 para transformar o  número em multiplo de 3 e vencer
